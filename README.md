# Swag Labs (SauceDemo) – Manual Testing Project

**Live Website**: https://www.saucedemo.com  
**Tester**: Mallikanti Usha  
**GitHub**: mallikanitusha2002-boop  
**Date**: November 2025
## Project Overview
Complete end-to-end manual testing of Swag Labs e-commerce demo  
Covered: Login → Product Catalog → Shopping Cart → Checkout → Logout  
- 42 detailed test cases using ECP & BVA  
- Found and documented **6 real defects** (including missing search bar, no ZIP validation, etc.)
